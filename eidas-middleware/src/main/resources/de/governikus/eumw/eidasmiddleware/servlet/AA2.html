Bitte beachten Sie die folgenden Hinweise:

Wie kann ich mich online ausweisen? Kurz erklärt...

Um die Online-Ausweisfunktion zu nutzen, müssen Sie folgende Voraussetzungen schaffen:

  1. Online-Ausweisfunktion aktivieren
  2. Kartenlesegerät bestellen / mobiles Endgerät ggf. per NFC-Verbindung möglich (siehe Liste)
  3. Transport-PIN in persönliche PIN ändern
  4. AusweisApp2 installieren und starten
Achtung: zum Online-Ausweisen benötigen Sie immer eine Internetverbindung. Die Details können Sie hier nachlesen.

Please take note of the following guideline:

How can I identify myself online - briefly explained...

To use the online identification function, you have to create the following preconditions:

  1. Activate online identification function
  2. Have a card reader or applicable mobile device ready
  3. Replace Transport-PIN with personal PIN
  4. Have the AusweisApp2 installed and running
Note: You always need an Internet connection for online identification. For more information click here.
